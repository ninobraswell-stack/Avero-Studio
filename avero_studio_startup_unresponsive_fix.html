Update Log

Current build notes for this file.

Current patch

Profile + accent update

  1. Replaced the old circular color wheel with a left-to-right RGB hue bar and a draggable circle thumb.
  2. Added a Reset Accent button that instantly restores the default interface color.
  3. Added profile customization with username editing, bio editing, and faded wallpaper presets.
  4. Added a live preview card so profile changes can be seen before saving.
  5. Added a bio line and wallpaper support to the welcome card.
  6. Added this full-screen update log panel inside settings.
  7. Optimized recent-message and group-composer loading to use cached conversation metadata first instead of re-reading every message thread on each screen change.
  8. Synced profile changes back into stored conversation member metadata so names and avatars refresh faster after edits.

Historical note

About older changes

  1. This file did not already contain a dated patch-note system, so this panel starts by documenting the current patch precisely.
  2. Existing features already present in the file before this patch include account creation, login, profile pictures, direct messages, group chats, reactions, replies, speech-to-text, light and dark mode, and glass-style performance modes.
